# ✰ Lost Moon ˚☽˚.⋆

A personal website with a night aesthetic, built with pure HTML and CSS.

## Purpose

This project exists for two simple reasons:

1. **Distraction** — a fun side project to work on between other, more serious projects.
2. **Learning** — a hands-on way to practice and improve HTML and CSS skills.

There are no big goals here. No frameworks, no libraries, no pressure. Just a cozy corner of the internet to experiment and have fun.

## What's being practiced

- HTML structure and semantics
- CSS layouts (Flexbox and Grid)
- CSS animations and effects
- Positioning elements (absolute, relative, z-index)
- Google Fonts integration
- Multi-page navigation
- Y2K/Neocities visual aesthetic

## Stack

- HTML
- CSS
- Vanilla JavaScript

## Inspiration

Old-school personal websites from the early 2000s — the kind you'd stumble upon as a kid and spend hours exploring.
